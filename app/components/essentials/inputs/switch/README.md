### On/off states
```js
<div>
  <Switch />
  <Switch checked="true" />
</div>
```

### Disabled state
```js
<div>
  <Switch disabled={true} />
  <Switch disabled={true} checked="true" />
</div>
```

### With label
```js
<div>
  <Switch label="Labeled switch" />
  <Switch label="Labeled switch" checked="true" />
</div>
```
